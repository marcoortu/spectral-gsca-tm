1 Introduction

egscatm fits a Structural Topic Model in Aitchison geometry via a closed-form spectral solution. Document–topic proportions live on the probability simplex \(\mathcal{S}^K\) and are linked to document-level covariates through a Generalized Structured Component Analysis (GSCA) formulation. Estimation reduces to a single eigendecomposition—no iterative algorithm, no random initialisation, no local optima.

This vignette walks through a complete analysis of poliblog5k, a corpus of 5 000 US political blog posts collected during the 2008 presidential campaign and distributed with the stm package (Roberts, Stewart & Tingley, 2019).

1.1 The ILR-EGSCA model in brief

For \(M\) documents, \(N\) terms, \(K\) topics and \(P\) covariates:

Symbol Dimension Meaning
\(\mathbf{W}\) \(M \times N\) Document–term matrix
\(\mathbf{C}\) \(M \times P\) Covariate matrix (column-centred)
\(\mathbf{\Pi}\) \(M \times K\) Topic proportions (rows \(\in \mathcal{S}^K\))
\(\boldsymbol{\Phi}\) \(K \times N\) Topic–term loadings
\(\mathbf{B}_z\) \(P \times (K{-}1)\) Covariate path coefficients in ILR space

The model solves

\[ \min_{\mathbf{Z},\boldsymbol{\Psi},\mathbf{B}_z,\boldsymbol{\mu}} \left\| \widetilde{\mathbf{W}} - \tfrac{1}{K}\,\mathbf{Z}\boldsymbol{\Psi} \right\|_F^2 + \lambda\,\left\|\mathbf{Z} - \mathbf{C}\mathbf{B}_z\right\|_F^2, \qquad \text{s.t.}\; \mathbf{Z}^\top\mathbf{Z} = \mathbf{I}_{K-1}, \]

where \(\mathbf{Z}\) are the ILR topic scores and \(\widetilde{\mathbf{W}}\) is the row-centred document–term matrix. After profiling out \(\boldsymbol{\Psi}\), \(\mathbf{B}_z\) and \(\boldsymbol{\mu}\), the problem reduces to the top-\((K{-}1)\) eigendecomposition of the \(M \times M\) matrix

\[ \mathbf{S}_z = \underbrace{\widetilde{\mathbf{W}}\widetilde{\mathbf{W}}^\top}_{\text{centred doc similarity (LSA)}} + \lambda\,\underbrace{\mathbf{C}(\mathbf{C}^\top\mathbf{C})^{-1}\mathbf{C}^\top}_{\text{covariate projector } \mathbf{P}_C}. \]

\(\lambda = 0\) recovers centred LSA; \(\lambda \to \infty\) collapses topics onto the covariate space.

3.3 Building the document–term matrix for egscatm

egscatm expects a dense numeric matrix \(\mathbf{W} \in \mathbb{R}_{\ge 0}^{M \times N}\). We convert stm’s sparse list format:

stm_to_dtm <- function(docs, V) {
  W <- matrix(0.0, length(docs), V)
  for (i in seq_along(docs)) {
    d <- docs[[i]]
    W[i, d[1L, ]] <- d[2L, ]
  }
  W
}

W <- stm_to_dtm(docs, V)
cat(sprintf("DTM: %d x %d,  sparsity %.1f%%\n",

4 Fitting the model

egscatm() requires only \(\mathbf{W}\), \(\mathbf{C}\), the number of topics \(K\), and the regularisation weight \(\lambda\). The default scale_W = TRUE row-normalises \(\mathbf{W}\) to term frequencies before fitting.

K      <- 7L
lambda <- 3.0

t_start <- proc.time()
fit <- egscatm(W, C, K = K, lambda = lambda, scale_W = TRUE)
t_fit   <- proc.time() - t_start

cat(sprintf("Fit completed in %.2f seconds (elapsed)\n", t_fit["elapsed"]))
#> Fit completed in 5.11 seconds (elapsed)

5.2 ILR topic–term deviation matrix

The matrix \(\hat{\boldsymbol{\Psi}} \in \mathbb{R}^{(K-1) \times N}\) captures topic-specific word deviations in ILR coordinates. Each row is the ILR projection of one direction; positive entries indicate words over-represented relative to the corpus baseline in that direction.

6 Document–topic proportions

6.1 Distribution across the corpus

fit$Pi is an \(M \times K\) matrix; each row sums to 1 by construction (the closure map \(\text{softmax}(\mathbf{V}\mathbf{z}_i^*)\) is the exact inverse of the ILR transformation, not an approximation).

cat("Row-sum range:", range(round(rowSums(fit$Pi), 10)), "\n")
#> Row-sum range: 1 1
cat("Min proportion:", min(fit$Pi), "\n")
#> Min proportion: 0.1229359
boxplot(
  fit$Pi,
  names  = paste0("T", seq_len(K)),
  col    = hcl.colors(K, "Set 2", alpha = 0.7),
  border = hcl.colors(K, "Set 2"),
  ylab   = "Topic proportion",
  main   = "Distribution of document–topic proportions",
  las    = 1
)
abline(h = 1 / K, lty = 2, col = "grey50")
legend("topright", "1/K centroid", lty = 2, col = "grey50", bty = "n",
       cex = 0.8)

The median of each topic is close to \(1/K \approx 0.14\) — this is expected: the ILR centroid \(\mathbf{z} = \mathbf{0}\) maps exactly to the uniform composition, so the average topic proportion is \(1/K\) by construction. The spread reflects how strongly individual documents deviate from the centroid.

7 Covariate effects

7.1 ILR path coefficients \(\hat{\mathbf{B}}_z\)

\(\hat{\mathbf{B}}_z \in \mathbb{R}^{P \times (K-1)}\) gives the OLS regression of the ILR topic scores on the covariates. Column \(k\) describes how covariate \(p\) shifts the topic composition along the \(k\)-th ILR direction.

Bz_df <- as.data.frame(round(fit$Bz, 4))
rownames(Bz_df) <- c("Liberal (vs Conservative)", "day (centred)")
colnames(Bz_df) <- paste0("ILR", seq_len(K - 1L))

The Liberal coefficient on ILR1 (−0.063) is the largest signal: Liberal blogs score systematically lower on the first ILR direction. Since ILR1 contrasts the first topic against the others, this reflects a shift in topic composition between political leanings.

10.1 Key properties

  • Non-iterative: the solution is the top-\((K{-}1)\) eigenspace of an \((r + P) \times (r + P)\) matrix (\(r \le \min(M, N, 100)\)) — no random initialisation, no convergence issues, fully deterministic.
  • Globally optimal: for the profiled, linearised ILR-EGSCA objective (Ky Fan theorem).
  • Simplex-consistent: topic proportions satisfy \(\hat{\pi}_{ik} \ge 0\) and \(\sum_k \hat{\pi}_{ik} = 1\) by construction — not through post-hoc clipping.
  • Compositional: the structural penalty \(\lambda \|\mathbf{Z} - \mathbf{C}\mathbf{B}_z\|_F^2\) equals the sum of squared Aitchison distances between observed and covariate-predicted topic compositions (isometry of ILR).
